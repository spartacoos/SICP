* Chapter 1 — Building Abstractions with Procedures

** Hello SICP

This block uses =#lang sicp= via ob-racket's =:lang sicp= header.

#+begin_src racket :lang sicp
(define (square x)
  (* x x))

(square 5)
#+end_src

** Sum of Squares

#+begin_src racket :lang sicp
(define (square x)
  (* x x))

(define (sum-of-squares x y)
  (+ (square x) (square y)))

(define (f a)
  (sum-of-squares (+ a 1) (* a 2)))

(f 5)
#+end_src

Expected result:

#+begin_example
136
#+end_example

* Pictures

** Simple Inline Picture With Racket pict

This is the reliable path for rendering pictures inside Org.

#+begin_src racket :lang racket :results file graphics :file images/hello-pict.png
(require pict
         file/convertible)

(define drawing
  (hc-append
   24
   (circle 80)
   (rectangle 120 80)
   (text "SICP" null 32)))

(send (pict->bitmap drawing) save-file "images/hello-pict.png" 'png)

"images/hello-pict.png"
#+end_src

After running the block, toggle inline images:

#+begin_example
C-c C-x C-v
#+end_example
